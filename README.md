## 搬瓦工 Login 流程：官网客户中心登录步骤

**搬瓦工的后台分两个部分：** 一个是官网"客户中心"（Client Area），用来查看和管理你的订单、账户；另一个是"KiwiVM 控制面板"，用来管理具体那台 VPS（重启、重装系统、切换机房等）。

### 登录客户中心（Client Area）

1. 打开 `bwh81.net`（或其他可用的镜像域名）
2. 点击右上角的 **Client Area**
3. 输入注册时的邮箱和密码
4. 点击 **Login**

登录后，你会看到自己购买的 VPS 列表，账单信息，以及提交工单的入口。

如果登录时出现 Cloudflare 验证或者密码提示错误，刷新再试一次。这是 Cloudflare 防护层的正常触发，不是账号有问题。

## KiwiVM 面板登录：管理 VPS 的核心操作

KiwiVM 是搬瓦工自研的控制面板，地址是 `kiwivm.64clouds.com`。所有 VPS 的实际操作——开机关机、重装系统、查看 IP、切换机房——都在这里。

### 方法一：从客户中心跳转（推荐）

这是最简单的方式，不需要记额外密码：

1. 登录官网客户中心
2. 点击顶部菜单 **Services → My Services**
3. 找到你的 VPS，点击进入详情页
4. 点击 **KiwiVM Control Panel** 按钮，自动跳转登录

整个过程不需要输入任何 KiwiVM 密码，系统自动完成验证。

### 方法二：直接访问 KiwiVM 地址

直接打开 `https://kiwivm.64clouds.com`，然后：

- 在 **VPS IP address** 栏填入你的服务器 IP
- 在 **KiwiVM API Key** 栏填入 KiwiVM 面板的独立密码

注意：这里的密码**不是**你登录官网的账号密码，也**不是** SSH 的 root 密码，是单独的 KiwiVM 面板密码。如果没设置过，可以在方法一进入 KiwiVM 面板后，左侧菜单找到 **KiwiVM password modification** 自行设置。

## KiwiVM 面板能做什么？核心功能一览

进了 KiwiVM 之后，左侧菜单里的功能不少，新手容易看晕。重点记几个：

**Main controls**：查看 VPS 当前状态（运行/停止）、IP 地址、流量使用量，执行重启和关机。

**Install new OS**：一键重装系统，支持 20+ 个 Linux 发行版，包括 Ubuntu、Debian、CentOS 等。重装后会显示新的 SSH 端口和 root 密码，务必保存。

**Migrate to another DC**：迁移机房。买了 CN2 GIA-E 套餐之后，从这里可以切换到其他机房，完全免费，通常几分钟搞定。

**Root password modification**：重置 VPS 的 root 密码。

**Two-factor authentication**：开启两步验证，提升账户安全性。建议开启。

**KiwiVM password modification**：设置或修改 KiwiVM 面板的独立登录密码。
